# Bijdrager Aan de slag

Deze sectie is bedoeld voor ingenieurs en medewerkers die lokale installatiebegeleiding nodig hebben, plus de projectregels voor het veilig verzenden van wijzigingen.

## Eerste stappen

1. Kloon de repository.
2. Controleer het app-buildpad met `clickable desktop`.
3. Bekijk de technische documenten om de splitsing van QML, JavaScript en Python te begrijpen.
4. Bekijk de PR-richtlijnen voordat u wijzigingen doorvoert.

## Werken aan de documentatiesite

De documentatiesite maakt nu deel uit van de repository:

```bash
cd website
npm install
npm run start
```

## Bijdrageverwachtingen

Bijdragers moeten de documentatie bijwerken wanneer er wijzigingen zijn:

- gebruiker zichtbaar
- architectuur-zichtbaar
- setup-gerelateerd
- release-impact
